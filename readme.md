# Easy Command v1.0.0

![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)
[![Minecraft: 1.20.1](https://img.shields.io/badge/Minecraft-1.20.1-blue.svg)](https://minecraft.net)

本模组为 Minecraft Forge 玩家提供图形化命令面板，无需记忆复杂指令，一键执行自定义命令！

**支持跨世界持久化**：所有命令保存在全局配置文件中，适用于所有存档。

---

## 安装及使用

1. 下载 [最新 Release 的 `.jar` 文件](https://github.com/puragon/easy_command/releases)
2. 将其放入 Minecraft 的 `mods` 文件夹
3. 启动游戏
4. 游戏中按 <kbd>G</kbd> 键可打开界面
